Rules ForeignCc

A rule for building projects using the Ninja build tool

ninja

ninja(name, additional_inputs, additional_tools, alwayslink, args, build_data, copts, data, defines,
      deps, directory, env, lib_name, lib_source, linkopts, out_bin_dir, out_binaries, out_data_dirs,
      out_headers_only, out_include_dir, out_interface_libs, out_lib_dir, out_shared_libs,
      out_static_libs, postfix_script, targets, tool_prefix, tools_deps)

Rule for building external libraries with Ninja.
